Slate:چوارچێوەیەکی سەرچاوەکراوە بۆ دیزاینی دەستکاریکەری دەق
ad
Oct 20, 2020
١ خولەک
50 وشە

Slate چوارچێوەیەکی سەرچاوەکراوە، بە ئیلهام لە Draft.js بۆ دەستکاری دەق دیزاین کراوە، ئەم چوارچیوە دەستی گەشەپێدەران زۆر ئاوەڵا دەکا بۆ دڵخواز کردنی.

یەکێک لە خاڵە بەهێزەکانی ئەم چوارچێوە ئەوەیە کە تێیدا، هەموو شتێک بە شێوازی زیادکراوە پێناسە کراوە؛تەنانەت ناوک(کێرنێڵ) ئەسڵی.ئەمەش گەشەپێدەرانی هانداوە کە زۆرتر دەستکاری تێدا بدەن و دڵخوازی بکەن.
