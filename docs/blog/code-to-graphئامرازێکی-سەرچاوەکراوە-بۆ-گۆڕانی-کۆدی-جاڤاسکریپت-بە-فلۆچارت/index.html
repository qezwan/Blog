Code to Graph ئامرازێکی سەرچاوەکراوە بۆ گۆڕانی کۆدەکانی جاڤاسکریپت بە وێناکانی فلۆچارت.ئەم ئامرازە لە لایەن گەشەپێدەرێکی فەرەنسی بە ناوی Vincent Lecrubier وشێندراوە.

بۆ فامی باشتری ئەم ئامرازە نەخشەیەک کە بە زمانی چاڤاسکریپت (JS) نووسراوە نیشان دەدەین:

const myFunction = (x) => {
    if (x > 0) {
        return "ok"
    } else {
        throw "not ok"
    }
}
