بۆ گۆڕینی قەبارەی شاشە لە لینوکسی-ئێلێمێنتری ئێوە دەتوانن لە مەسیری خوارەوە ئەندازەی ڕوناکی شاسەی سیستەمەکەتان بگۆڕن.

System Settings >Displayes
